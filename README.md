# 情報セキュリティ関連の大学研究室Webサイトをまとめてみた
情報セキュリティを大学や大学院で学びたい研究したい、という進学希望の方に向けて。

Webページを持っていることが確認できた研究室だけ掲載していますので、このほかにもまだまだ研究室はあると思います。

客員の研究員や教員の方はお名前に入れていません。
